## SkillUpPlus 2030

O aplicativo SkillUpPlus2030+ é uma solução mobile que tem o objetivo de auxiliar usuários em transição de carreira e reespecialização (reskilling) em áreas demandadas pelo mercado, coletando informações sobre objetivos profissionais e perfil atual, para gerar recomendações de carreiras personalizadas usando IA.
